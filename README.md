# 3D 프린팅 역보정 프로그램 (Reverse Calibration Tool)

이 프로그램은 3D 프린팅 시 발생하는 수축 및 변형을 보정하기 위해, CAD 원본과 실제 출력물 스캔 데이터(또는 시뮬레이션 결과)를 비교하여 역보정된 모델과 수치 데이터를 생성합니다.

## 🛠 기능
1. **3D 파일 업로드:** CAD 원본, 예측 형상, 스캔 데이터 (STL, OBJ 지원).
2. **자동 정렬:** 무게중심 기준 자동 정렬 (ICP 정밀 정렬 확장은 코드 내 주석 참조).
3. **오차 분석:** CAD 표면 기준 변형량(Distance) 계산 및 3D 히트맵 시각화.
4. **역보정 결과:**
   - **보정된 STL 파일:** 변형의 반대 방향으로 Vertex를 이동시킨 모델 다운로드.
   - **CSV 데이터:** 각 지점별 X, Y, Z 방향의 오차 및 보정값 다운로드.

## 📦 설치 방법

Python이 설치되어 있어야 합니다.

1. 필수 라이브러리 설치:
   ```bash
   pip install -r requirements.txt
   ```

## 🚀 실행 방법

다음 명령어로 웹 인터페이스를 실행합니다:

```bash
streamlit run app.py
```

실행 후 브라우저가 자동으로 열리며, 로컬 주소(보통 `http://localhost:8501`)로 접속됩니다.
